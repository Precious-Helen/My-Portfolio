# Portfolio Project

## Overview
This portfolio project showcases the work and skills of Helen, a web developer. It includes a main HTML file (`Port.html`) that presents the portfolio content, along with a CSS file (`assets/body.css`) that styles the webpage.

## Files
- **Port.html**: The main HTML document containing the structure and content of the portfolio.
- **assets/body.css**: The stylesheet that defines the layout, colors, fonts, and other visual aspects of the HTML content.

## Setup Instructions
1. Clone the repository to your local machine using:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd portfolio-project
   ```
3. Open `Port.html` in your web browser to view the portfolio.

## Usage
Feel free to modify the content in `Port.html` and the styles in `assets/body.css` to personalize the portfolio. You can add more sections, images, or styles as needed.
